iPhone / iPad

  1. Abre el enlace en Safari (no Chrome).
  2. Toca Compartir → Añadir a pantalla de inicio.
  3. Entra con JOM / Studio.
  4. Permite ubicación y Bluetooth si los pide.

Android

  1. Abre el enlace en Chrome.
  2. Menú ⋮ → Instalar aplicación o Añadir a inicio.
  3. O instala el APK incluido en el zip (si está compilado).
